# 村圳首批内容脚本

更新日期：2026-03-09

## 1. 使用方式

这批脚本是为“低时间投入、快速试错”设计的。

你不需要拍很复杂，只要做到：

- 一个脚本只讲一个点
- 开头 `3` 秒先报预算 / 地点 / 人群
- 结尾只给一个 CTA
- 每条内容都能把人导向私信或表单

## 2. 脚本 1：西乡 1200 单间实拍

### 标题

`西乡上班，预算 1200，这种单间今天还能看`

### 开场口播

“在西乡附近上班，预算 `1200` 左右，今天还有这种单间能看，我带你直接过一遍真实情况。”

### 镜头顺序

- 门口和楼道
- 房间全景
- 窗户和采光
- 卫生间
- 水电和押付说明

### 结尾 CTA

“你如果也在西乡上班，把 `预算 + 上班地` 私信我，我给你配 `3` 套今天还能看的。”

## 3. 脚本 2：福永 1500 一房实拍

### 标题

`福永 1500 左右能租到什么样的一房`

### 开场口播

“很多人以为 `1500` 在福永租不到像样的一房，这套我今天刚看，给你看真实房态。”

### 镜头重点

- 客厅 / 卧室分区
- 厨卫
- 楼层和采光
- 周边环境一句话说明

### 结尾 CTA

“想看同预算的一房，直接发我 `1500 + 福永 + 一房`。”

## 4. 脚本 3：机场附近上班住哪里

### 标题

`在机场附近上班，1000 到 1600 住哪几个村最省钱`

### 内容结构

- 先给人群：机场附近上班
- 再给预算带：`1000-1600`
- 再推荐 `2-3` 个村
- 每个村说一个优点和一个限制

### 结尾 CTA

“你把 `上班地 + 预算` 发我，我按通勤和预算给你配村。”

## 5. 脚本 4：西乡 1800 怎么选村

### 标题

`西乡上班，预算 1800，住哪几个村最稳`

### 口播提纲

- 如果你通勤优先，选哪里
- 如果你想性价比高，选哪里
- 如果你更怕夜路和环境差，选哪里

### 结尾 CTA

“我最近就在筛西乡这条线，私信 `1800 + 西乡`，我给你直接配房。”

## 6. 脚本 5：单间看房先看哪 5 个点

### 标题

`城中村单间看房，先看这 5 个点，不然很容易踩坑`

### 5 个点

- 水电怎么算
- 采光和通风
- 卫生间有没有异味
- 晚上回家路上安不安全
- 楼道和隔音情况

### 结尾 CTA

“如果你不想自己一家家踩坑，我会先把不合格的过滤掉。”

## 7. 脚本 6：水电规则怎么看

### 标题

`深圳城中村租房，最容易吃亏的就是水电`

### 口播提纲

- 不要只看房租
- 问清楚水电单价
- 问清楚还有没有管理费、卫生费、网费
- 费用说不清的房源我直接不推

### 结尾 CTA

“你看房前不知道该问什么，就发我‘水电’，我给你一份提问清单。”

## 8. 脚本 7：夜归女生怎么选

### 标题

`晚上下班晚的女生，城中村看房先看哪几点`

### 口播提纲

- 主路和巷道照明
- 楼道和门口环境
- 回家最后 `200` 米是不是太偏
- 是否有电梯和门禁

### 结尾 CTA

“如果你是晚归租客，发我 `女生 + 预算 + 上班地`，我只给你筛相对稳的房。”

## 9. 脚本 8：什么房源我不会发

### 标题

`这种房源哪怕便宜，我也不会发`

### 不发标准

- 没视频
- 价格说不清
- 水电不透明
- 超过 `72` 小时没刷新
- 明显跟描述不一致

### 结尾 CTA

“我宁愿房源少一点，也不想让你白跑。”

## 10. 脚本 9：一房和单间差价值不值

### 标题

`单间和一房到底差多少钱，值不值得多花`

### 口播提纲

- 常见差价区间
- 什么人适合直接上一房
- 什么人先住单间更划算

### 结尾 CTA

“你告诉我预算和通勤，我帮你判断单间还是一房更值。”

## 11. 脚本 10：固戍 / 后瑞对比

### 标题

`固戍和后瑞，哪个更适合预算 1500 左右的人`

### 对比维度

- 通勤
- 村内环境
- 房型选择
- 夜间回家感受

### 结尾 CTA

“你在这两边上班的，直接发我预算，我给你具体配。”

## 12. 脚本 11：最近最好租的 3 类房源

### 标题

`最近最容易成交的 3 类房源，房东也可以看看`

### 内容结构

- 预算带一
- 预算带二
- 房型 / 片区组合

### 作用

这条既能吸引租客，也能吸引供给侧。

## 13. 脚本 12：集中带看通知

### 标题

`本周三 / 周六，西乡-福永集中带看，预算 1000 到 2000 的可以来`

### 文案

“这周我集中带看一批西乡 / 固戍 / 福永的单间和一房，预算 `1000-2000` 的，想省时间可以直接跟一趟。把 `预算 + 上班地 + 房型` 发我，我确认后给你排进去。”

## 14. 图文版通用模板

### 模板结构

- 第一行：预算 + 地点 + 人群
- 第二行：你能解决什么问题
- 第三到五行：给真实信息
- 最后一行：一个 CTA

### 示例

“西乡上班，预算 `1500` 左右，如果你不想自己一家家跑，我这边最近在筛一批真房源。
今天这套一房，优点是采光和通勤都还可以，缺点是楼层一般。
我只发有视频、费用说得清、最近核验过的房。
想看同预算的，私信我：`1500 + 西乡 + 一房`。”

## 15. 发布纪律

- 不追求剪辑炫技
- 不追求一次讲很多信息
- 不追求所有人都喜欢
- 只追求：有人私信、有人填表、有人预约
