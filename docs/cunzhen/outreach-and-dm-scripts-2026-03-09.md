# 村圳邀约与私信脚本

更新日期：2026-03-09

## 1. 供给侧首次邀约

“我现在在做宝安西乡 / 福永这条线的租房成交服务，主要筛高意向租客，不收租客钱。你这边如果有真实在租、视频齐全、费用说得清的房源，我可以先帮你带看，成交后再结算服务费。”

## 2. 供给侧跟进

“我这边不会把纯问价的人都塞给你，我会先筛预算、上班地和入住时间。你如果愿意配合补视频和更新时间，我会优先给你推高意向客户。”

## 3. 租客首次回复

“收到，你把 `预算 + 上班地 + 房型 + 最晚入住时间` 发我，我按你的情况先给你配 `3` 套最近还能看的。”

## 4. 推房后跟进

“这 `3` 套你更偏哪一套？如果你想省时间，我给你安排到最近一场集中带看。”

## 5. 催表单补全

“你顺手把求租卡补一下，后面我这边有新房源就能直接按条件发你，不用每次重新说。”

## 6. 带看前确认

“今晚 / 明天带看我这边已经排上了。你再确认一下时间、预算和想看的房型，避免到场后浪费你时间。”

## 7. 爽约后回收

“今天没碰上没关系，我这边先记一下原因。你是时间不合适，还是房型 / 预算要调整？我后面按新条件再给你筛。”

## 8. 成交后向供给侧结算

“这单已经明确成交，我这边按之前说好的固定服务费结算。后面如果你还有同类型真房源，我优先继续帮你带高意向客户。”
